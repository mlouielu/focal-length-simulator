Do I need more focal length? Try out with your own photos!

Motivation

As an amateur photographer, I often find myself capturing bird photos with my standard lens, only to see the bird appearing quite small in the frame. This leads me to a recurring question after each shot:

What would be the ideal focal length to make these photos look perfect, or at least better?

While there are numerous focal length simulators available online, none allow the use of personal photographs for simulation.

So I decided to build one, and this is it! The only tool that allows for the use of your own photos to determine the best focal length for a shot.

How To Use

Basic

The red bounding box indicates the photo size of the current focal length, and it should be the same at the beginning.

You can drag, zoom in, and zoom out to interact with and simulate different focal lengths within the photo.

Use Your Own Photo

It is completely offline and local! Nothing will be uploaded to any cloud or server.

Click "Choose File" and select the photo you wish to simulate. The current photo will be replaced with the one you selected.

This tool attempts to retrieve the lens focal length from your photo's EXIF data. If this fails, you will need to input it manually.

If you input the lens focal length manually or encounter any issue, click "Reset Zoom" to reset the red bounding box.

How To Zoom In

There are five ways to zoom in:

  1. Double Tap
  2. Gesture Zoom
  3. Wheel Scrolling
  4. Range Bar
  5. Manual Enter
